git clone — initially creates the repository. only need to do once


git checkout -b “branch_name” — create new branch

git commit -am “new commits”

git pull origin master — brings in changes others have made

git push origin branch_name — have to still merge online

ADDING CHANGES
git checkout master
git pull origin master
git merge <branch_name>
git push origin master

git branch -d // delete a branch locally (not online)
